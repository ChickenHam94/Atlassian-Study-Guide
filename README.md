# ACA-925 Prep Platform v2

A polished GitHub Pages-ready certification prep platform for Atlassian Jira Service Management Agent Essentials.

## Features

- 500+ original practice questions
- Custom practice sessions
- Timed mock exams
- Adaptive weak-area practice
- Flashcards
- Question bank browser
- Results dashboard
- Dark mode
- Android-ready PWA
- Offline caching over HTTPS
- GitHub Pages compatible

## Deploy to GitHub Pages

1. Upload these files to your repository root.
2. Go to **Settings -> Pages**.
3. Choose **Deploy from a branch**.
4. Select **main** and **/root**.
5. Save.

Your site will be:

```text
https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/
```

## Android install

After the site is live on GitHub Pages, open it in Android Chrome or Edge and choose **Install app** or **Add to Home screen**.

## Content note

This project uses original practice questions only. It does not contain real/live Atlassian exam questions or copied exam dumps.
